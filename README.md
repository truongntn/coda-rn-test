## Project `Coda-rn-test`

#### Run project

- Run `npm install` to install libraries
- Run `expo start` to automatically configure TypeScript
- Choose `Run on Android device/emulator` or `Run on iOS simulator`

### Tech stack

- Expo
- Firebae/Firestore
